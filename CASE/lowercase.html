Lower CASE vahendid

Definitsioon

Tegemist on teostusekesksete, mudelitest reaalse tarkvaratoote loomise käigus kasutatavate vahenditega.

Mille jaoks need arenduses kasulikud on:

  • Andmebaasi struktuuri genereerimine
  • Koodi genereerimine
  • Testide läbiviimine
  • Koodi versioonihaldus
  • Konfiguratsioonihaldus
  • Pöördprojekteerimine

Lower CASE vahendid on enamasti seotud backend osaga tarkvaraarendusest ning levinumad tööriistad on koodigenereerijad, kompileerijad ja testimist hõlbustavad tööriistad.

Isiklik kogemus

Kuna olen juba paar aastat tegelenud Java arendusega, olen paljude Lower CASE tööriistade esindajatega kokku puutunud. Raske on öelda täpselt, millistega ma pole kokku puutunud, sest ma ei tea kõiki. Olen peamiselt kokku puutunud IDE-dest IntelliJ, Visual Studio Code ja Visual Studioga. Kasutan ka Git version controli. Arenduse käigus testimisel kasutan palju ka silumistööriistu (debugger) IntelliJ-siseselt.
